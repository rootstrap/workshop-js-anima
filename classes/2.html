Variables

Las variables son contenedores que nos permiten almacenar valores. Para declarar una variable podemos utilizar la palabra clave let. 

						let myVariable;
También podemos asignarle un valor al momento de su declaración. 

						let myVariable = "a value";
JavaScript es un lenguaje dinámico y de tipado flexible.
Las variables en JavaScript no están asociadas directamente con ningún tipo de valor en particular, y a cualquier variable se le pueden asignar (y reasignar) valores de todos los tipos. 

						let foo = 42;    // foo es un número
						foo     = 'bar'; // foo es un string
						foo     = true;  // foo es un boolean
Además de variables, podemos definir constantes.
A diferencia de las variables, una constante no puede cambiar su valor posteriormente, por lo que se debe especificar su valor en la misma sentencia en la que se declara. 

						const foo = 7; // foo es una constante con el número 7
						foo = 20;      // lanzará un error

Valores

Todos los lenguajes de programación tienen estructuras de datos integradas, pero éstas a menudo difieren de un lenguaje a otro.
JavaScript posee 6 tipos de datos primitivos
Undefined
Una variable a la que no se le ha asignado un valor tiene el valor indefinido 

						let myVariable;
						console.log(myVariable); # imprime undefined
Boolean
Representa una entidad lógica y puede tener dos valores: verdadero y falso 

						let myVariable1 = true;
						console.log(myVariable1); # imprime el valor booleano true

						let myVariable2 = false;
						console.log(myVariable2); # imprime el valor booleano false
Number
Números entre -(253 − 1) y 253 − 1.
Adicionalmente, JS cuenta con 3 tipos simbólicos: +Infinity, -Infinity, y NaN ("Not a Number"). 

						let myVariable = 7;
						console.log(myVariable); # imprime el número 7
String
Sirve para representar una cadena de caracteres. La longitud del string está dada por la cantidad de elementos que lo componen. 

						let myVariable = "Hola mundo!";
						console.log(myVariable); # imprime Hola mundo!
BigInt
Un BigInt se crea agregando n al final de un número entero o llamando al constructor. Nos permite representar y operar con números enteros largos, que excedan los límites de Number. 

						let hugeNumber = BigInt(9007199254740991);
						console.log(hugeNumber); // imprime 9007199254740991n
						
						let alsoHuge = 2n ** 53n;
						console.log(alsoHuge); // imprime 9007199254740992n
Symbol
Valor único e inmutable que puede ser usado como clave para acceder a objetos. 

						let Sym1 = Symbol("Sym");
						console.log(Sym1); // imprime Symbol(Sym)
Existe además un tipo primitivo especial, llamado null.
Este valor se utiliza para representar la no existencia o invalidez de algo, generalmente de manera intencional.
